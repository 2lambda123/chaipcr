PUT /stages/:id
Update a stage

Supported Formats

json

Examples

{'stage':{'id':1,'stage_type':'holding','name':'Holding Stage','num_cycles':1}}

Params

Param Name Description
stage
required

Stage Info


Value: Must be a Hash
stage[stage_type]
required

Stage type


Value: Must be one of: holding, cycling, meltcurve.
stage[name]
optional

Name of the stage, if not provided, default name is '<stage type> Stage'


Value: Must be String
stage[num_cycles]
optional

Number of cycles in a stage, must be >= 1, default to 1


Value: Must be Integer
