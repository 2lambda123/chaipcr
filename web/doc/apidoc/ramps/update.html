PUT /ramps/:id
Update a ramp

Supported Formats

json

Examples

{'ramp':{'id':1,'rate':'100.0','max':true}}

Params

Param Name Description
ramp
required

Ramp Info


Value: Must be a Hash
ramp[rate]
required

Rate of the ramp, in degrees C/s, set to 100 for max, precision to 8 decimal point


Value: Must be Float
