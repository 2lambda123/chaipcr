PUT /experiments/:id
Update an experiment

Supported Formats

json

Examples

{'experiment':{'id':1,'name':'test','qpcr':true,'started_at':null,'completed_at':null,'completed_status':null}}

Params

Param Name Description
experiment
required

Experiment Info


Value: Must be a Hash
experiment[name]
required

Name of the experiment


Value: Must be String
experiment[qpcr]
optional

Define whether it is an pcr or qpcr experiment


Value: Must be 'true' or 'false'
