Experiments

>>>

GET /experiments
List experiments and show the first one

>>>

GET /experiments/new
New an experiment

>>>

POST /experiments
Create an experiment

when experiment is created, default protocol will be created

Params

Param name Description
experiment
required

Experiment Info


Value: Must be a Hash
experiment[name]
required

Name of the experiment


Value: Must be String
experiment[qpcr]
optional

Define whether it is an pcr or qpcr experiment


Value: Must be 'true' or 'false'
