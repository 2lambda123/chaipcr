Experiments / create

POST /experiments
Create an experiment

when experiment is created, default protocol will be created

Examples

{'experiment':{'id':1,'name':'test','qpcr':true,'started_at':null,'completed_at':null,'completed_status':null,'protocol':{'id':1,'lid_temperature':'110.0','stages':[{'stage':{'id':1,'stage_type':'holding','name':'Holding Stage','num_cycles':1,'steps':[{'step':{'id':1,'name':'Step 1','temperature':'95.0','hold_time':180,'ramp':{'id':1,'rate':'100.0','max':true}}}]}},{'stage':{'id':2,'stage_type':'cycling','name':'Cycling Stage','num_cycles':40,'steps':[{'step':{'id':2,'name':'Step 2','temperature':'95.0','hold_time':30,'ramp':{'id':2,'rate':'100.0','max':true}}},{'step':{'id':3,'name':'Step 2','temperature':'60.0','hold_time':30,'ramp':{'id':3,'rate':'100.0','max':true}}}]}},{'stage':{'id':3,'stage_type':'holding','name':'Holding Stage','num_cycles':1,'steps':[{'step':{'id':4,'name':'Step 1','temperature':'4.0','hold_time':0,'ramp':{'id':4,'rate':'100.0','max':true}}}]}}]}}}

Params

  • experiment : required [ Must be a Hash ]

    Experiment Info

    • name : required [ Must be String ]

      Name of the experiment

    • qpcr : optional [ Must be 'true' or 'false' ]

      Define whether it is an pcr or qpcr experiment

Experiments / temperature_data

GET /experiments/:id/temperature_data?starttime=xx&endtime=xx&resolution=xx
Retrieve temperature data

Params

  • starttime : required [ Must be Integer ]

    0 means start of the experiment, in ms

  • endtime : optional [ Must be Integer ]

    if not specified, it returns everything to the end of the experiment, in ms

  • resolution : optional [ Must be Integer ]

    Include data points for every x milliseconds. Must be a multiple of 1000 ms

Stages / create

POST /protocols/:protocol_id/stages
Create a stage

Examples

{'stage':{'id':1,'stage_type':'holding','name':'Holding Stage','num_cycles':1,'steps':[{'step':{'id':1,'name':'Step 1','temperature':'95.0','hold_time':180,'ramp':{'id':1,'rate':'100.0','max':true}}}

Params

  • stage : required [ Must be a Hash ]

    Stage Info

    • stage_type : required [ Must be one of: holding, cycling, meltcurve. ]

      Stage type

    • name : optional [ Must be String ]

      Name of the stage, if not provided, default name is '<stage type> Stage'

    • num_cycles : optional [ Must be Integer ]

      Number of cycles in a stage, must be >= 1, default to 1

  • prev_id : optional [ Must be Integer ]

    prev stage id or null if it is the first node

Steps

Steps / create

POST /stages/:stage_id/steps
Create a step

if step is created with no params, it will be the same as previous step; when step is created, default ramp with max rate (100) will be created

Examples

{'step':{'id':1,'name':'Step 1','temperature':'95.0','hold_time':180,'ramp':{'id':1,'rate':'100.0','max':true}}}

Params

  • step : required [ Must be a Hash ]

    Step Info

    • temperature : optional [ Must be Float ]

      Temperature of the step, in degree C, with precision to one decimal point

    • hold_time : optional [ Must be Integer ]

      Hold time of the step, in seconds, 0 means infinite

    • name : optional [ Must be String ]

      Name of the step, if not provided, default name is 'Step <order_number>'

  • prev_id : optional [ Must be Integer ]

    prev step id or null if it is the first node
